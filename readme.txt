== Description ==

Impulse laat publishers moeiteloos extra geld verdienen met hun content. Met de installatie van slechts een klein stukje software, worden producten te koop aangeboden in lijn met de content en lay-out van de site. Zonder storende banners en zonder dat bezoekers de site verlaten. Impulse verzorgt ook het hele koopproces: van productaanbod, betaling en bezorging tot retourneren en klantenservice; publishers hoeven zelf niets extra’s te doen. En ze krijgen er nog voor betaald ook. Elke maand maakt Impulse het bedrag over dat publishers hebben verdiend met de verkopen via hun site. Impulse rekent af met storende banners en tijdrovende webshops. Impulse maakt cash van content.

Heb je interesse? Meld je dan eerst aan via https://info.impulse.click/#meerweten of stuur een mail naar info@impulse.click, dan helpen wij je graag verder!

= Hoe werkt het? =
Installeer de plugin van Impulse
Plaats de trigger op jouw site
Stel samen met ons het productaanbod samen
Impulse zorgt voor de rest en maakt iedere maand het geld naar je over

== Installation ==

= Hoe installeer ik de plugin? =
Via Wordpress.org (https://wordpress.org/):
1. Zoek op de website van Wordpress.org naar “Impulse”.
2. Download de plugin “Impulse” als zip-bestand.
3. Upload de plugin door in je Wordpress dashboard naar “Plugins” te gaan. Klik vervolgens op “Nieuwe plugin” en “Plugin uploaden”.
4. Upload het zip-bestand en druk op “Nu installeren”. Vervolgens druk je op “Activeren” om de installatie af te ronden.

Via Wordpress dashboard:
1. Zoek in de bibliotheek naar “Impulse”.
2. Klik op “Installeren”.
3. Activeer de plugin.

== Frequently Asked Questions ==

= Hoe installeer ik de plugin? =
Via Wordpress.org (https://wordpress.org/):
1. Zoek op de website van Wordpress.org naar “Impulse”.
2. Download de plugin “Impulse” als zip-bestand.
3. Upload de plugin door in je Wordpress dashboard naar “Plugins” te gaan. Klik vervolgens op “Nieuwe plugin” en “Plugin uploaden”.
4. Upload het zip-bestand en druk op “Nu installeren”. Vervolgens druk je op “Activeren” om de installatie af te ronden.

Via Wordpress dashboard:
1. Zoek in de bibliotheek naar “Impulse”.
2. Klik op “Installeren”.
3. Activeer de plugin.

= Waar kan ik de Impulse instellingen vinden? =
Instellingen kun je vinden en aanpassen door aan de linkerkant van het dashboard te klikken op “Impulse instellingen”.

= Wat is een Client ID? =
Je Client ID is een persoonlijke code die je via de mail van Impulse krijgt. Deze code typ je in het lege veld achter “Client ID” bij “Impulse instellingen”. Hier hoef je verder niets mee te doen. 

= Hoe verander ik de kleur van de widget header? =
Bij “Impulse instellingen” kun je op het kleurenpalet klikken om een kleur uit te kiezen, je kunt ook de html-code van de gewenste kleur invullen.
Tip: het is belangrijk dat deze kleur past bij de stijl van jouw site

= Hoe krijg ik mijn logo in de Impulse shop? =
Bij “Impulse instellingen” kun je op “Upload een nieuwe logo” klikken. Kies hier voor jouw eigen logo. Om je logo op te slaan druk je op “Wijzigingen opslaan”. Je logo wordt nu vertoond in de “widget header” van de Impulse shop. 
Tip: kies een logo met een transparante achtergrond

= Wat zijn standaard triggers? =
Wanneer je deze optie inschakelt, wordt er aan de bovenkant van elke pagina automatisch een trigger geplaatst. Deze functie kun je inschakelen bij “Impulse instellingen”. Op elke pagina/bericht zal dan een trigger getoond worden. De trigger kun je per bericht/pagina in- en uitschakelen aan de rechterkant van je scherm waar je je bericht/pagina maakt. 

= Hoe krijg ik mijn eigen triggers in een blog? =
Wanneer je een blog of artikel aan het bewerken bent zie je aan de bovenkant een extra button “Impulse toevoegen”. Hier zie je een aantal standaard triggers die je kunt gebruiken. Ook heb je hier de optie om een eigen trigger te uploaden. Dit doe je door te klikken op “Upload je eigen trigger”. Kies hier voor een foto of logo en klik op “Publiceren”. Als je deze trigger nu toevoegt aan je blog (“Voeg in tekst”) dan wordt deze klikbaar. 
Naast een afbeelding/icoon als trigger, kan er ook gebruik worden gemaakt van een hyperlink of button als trigger.

= Hoe pas ik het formaat en positie van de trigger aan? =
In het venster van “Impulse toevoegen” kun je kiezen om de trigger in het originele formaat, medium formaat (300px) of klein (90px) formaat te tonen. Ook is er de optie om de trigger links, recht of in het midden uit te lijnen.
